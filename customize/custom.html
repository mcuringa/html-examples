As our first exploration of javascript, we're going to learn how to use some basic programming concepts to allow users to customize a page. Study this code, and then after you have a good understanding, modify the code to complete each of these tasks:

  1. Add a new option for background image.
  2. Add a new music option.
  3. Add an option to change the font size.
  4. Add an option to change the font face.
  5. Change the music buttons so that they pause the song if it's already playing
  6. Add a mute/volume icon that shows if music is playing or not
  7. Add something cool that you thought of on your own
